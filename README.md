# 🚀 Petit Exercice Kotlin

Ce dépôt contient une série de petits exercices en Kotlin pour pratiquer les bases de la programmation (boucles, conditions, listes, entrées/sorties).

## 📂 Contenu du projet

Voici la liste des exercices inclus :

- **exercice.kt** : Vérification de la parité (Pair/Impair).
- **exercice2.kt** : Calcul de la somme de 5 nombres saisis.
- **exercice3.kt** : Calcul de la factorielle d'un nombre.
- **exercice4.kt** : Introduction aux boucles `for`.
- **exercice5.kt** : Compteur de nombres positifs (boucle `do...while`).
- **exercice6.kt** : Mini-jeu "Plus Grand ou Plus Petit".
- **exercice9.kt** : Manipulation de listes et calcul de somme.
- **exercice10.kt** : Manipulation de chaînes de caractères (taille et contenu).
- **exercice11.kt** : Filtrage des nombres impairs dans une liste.
- **exercice12.kt** : Recherche de la valeur maximale dans une liste.
- **exercice14.kt** : Menu interactif pour la gestion d'une liste (Ajouter, Supprimer, etc.).

## ⚠️ Notes importantes
Le code actuel contient des erreurs de syntaxe communes (comme `readLin` au lieu de `readLine`) car il s'agit d'un projet en cours d'apprentissage.
